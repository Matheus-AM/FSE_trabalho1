## Intruções
Para rodar o makefile, após clonar o repositório basta executar na raiz do diretório:

```
$ make
```

Em caso de erro do bin/bin do makefile, criar uma pasta bin e executar novamente.

Em seguida, executar:

```
$ bin/bin
```

Para finalizar:

```
$ Ctrl + c
```
